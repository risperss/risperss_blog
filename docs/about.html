An about page

This is an example application using YOCaml 2! The aim of this blog is to show you how to quickly build a full-featured blog based on the new version of YOCaml (using archetypes).

The fact that this is a sample blog justifies the violently generated (and completely uninteresting) articles. Sorry about that!

You can refer to the code of this project to see how to build a blog yourself using, for example, this template that allows you to bootstrap an empty project filled with YOCaml dependencies.
